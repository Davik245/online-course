# online-course
Данный проект представляет собой **монолитное backend-приложение**, разработанное на **Java + Spring Boot**, предназначенное для учета:
- курсов повышения квалификации;
- слушателей (студентов);
- преподавателей;
- записей на курсы;
- платежей и финансовых операций.

Проект реализует бизнес-логику учебного центра и предназначен для использования администраторами и преподавателями.

## Цель проекта
Разработка информационной системы, обеспечивающей:
- централизованное хранение данных;
- автоматизацию процессов записи на курсы;
- контроль платежей;
- формирование отчетности;
- снижение количества ошибок при ручном учете.

  ## Функциональные требования

### Управление слушателями (Students)
- регистрация слушателей;
- хранение персональных и контактных данных;
- просмотр истории обучения;
- просмотр платежей слушателя.

### Управление преподавателями (Teachers)
- добавление и редактирование преподавателей;
- назначение преподавателей на курсы;
- просмотр курсов преподавателя.

### Управление курсами (Courses)
- создание и редактирование курсов;
- указание стоимости, вместимости и преподавателя;
- просмотр списка слушателей курса;
- расчет выручки по курсу.

### Запись на курсы (Enrollments)
- запись слушателя на курс;
- контроль максимального количества слушателей;
- запрет повторной записи на один и тот же курс;
- изменение статуса записи.

### Платежи (Payments)
- регистрация платежей;
- контроль суммы платежа;
- учет даты платежа;
- поддержка статусов платежа (PAID / UNPAID);
- связь платежа с курсом и слушателем.

## Нефункциональные требования

- REST API архитектура;
- валидация входных данных;
- централизованная обработка ошибок;
- документирование API через Swagger;
- миграции базы данных через Liquibase;
- покрытие ключевой бизнес-логики тестами.
